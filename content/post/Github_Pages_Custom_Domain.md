*关于域名解析记录的分类*

| 类型    | 用途                           |
| ----- | ---------------------------- |
| A     | 用于将域名解析到 IPv4 address        |
| AAAA  | 用于将域名解析到 IPv6 address        |
| CNAME | 用于将域名解析到另一个域名 Canonical Name |
