# 0x01 Install

Download Tarball 版本 <br>

Tarball 版本安装：<br>

```
mv /opt
ln -s /opt/nvim/bin/nvim /usr/local/bin/nvim
```

为什么使用 Tarball 版本？<br>
方便管理。<br>
